# 📝 Todo List App

Сучасний та мінімалістичний Todo List застосунок, створений з використанням **чистого HTML, CSS та JavaScript** без жодних бібліотек чи фреймворків.

![Todo Preview](https://via.placeholder.com/1200x600.png?text=Todo+List+Preview)

---

## ✨ Можливості

- ➕ Додавання нових задач
- ✅ Позначення виконаних задач
- 🗑 Видалення задач
- 💾 Збереження даних у `localStorage`
- 📱 Адаптивний дизайн для мобільних пристроїв
- 🎨 Сучасний UI з плавними анімаціями
- 🌙 Легка та приємна кольорова схема

---

## 🚀 Технології

Проєкт створений за допомогою:

- **HTML5**
- **CSS3**
- **Vanilla JavaScript (ES6)**

Без використання:

- React
- Vue
- Bootstrap
- jQuery
